Hutool

Hutool是一个Java工具包，也只是一个工具包，它帮助我们简化每一行代码，减少每一个方法，让Java语言也可以“甜甜的”

talent-tan/t-io

t-io是基于aio(nio2)的网络编程框架，和netty属于同类，但t-io更注重一线开发工程师的感受，提供了大量和业务相关的API。基于t-io来开发IM、TCP私有协议、RPC、游戏服务器端

人人开源/renren-security

采用SpringBoot2.0、MyBatis、Shiro框架，开发的一套权限系统，极低门槛，拿来即用。设计之初，就非常注重安全性，为企业系统保驾护航，让一切都变得如此简单

Lemur开源/easypoi

POI 工具类,Excel的快速导入导出,Excel模板导出,Word模板导出,可以仅仅5行代码就可以完成Excel的导入导出,修改导出格式简单粗暴,快速有效,easypoi值得你尝试

Javen/IJPay

IJPay 让支付触手可及，封装了微信支付、支付宝支付、银联支付常用的支付方式以及各种常用的接口。不依赖任何第三方 mvc 框架，仅仅作为工具使用简单快速完成支付模块的开发，可轻松嵌入

jmdhappy/xxpay

XxPay聚合支付使用Java开发，包括spring-cloud、dubbo、spring-boot三个架构版本，已接入微信、支付宝等主流支付渠道，可直接用于生产环境

Cointop 控制台的加密货币跟踪工具

Cointop是一个快速而且轻量级的交互式终端应用，用来实时跟踪和监控加密货币行情。其灵感来自于 htop ，而键盘操作习惯来在于 vim

Akraino Edge Stack 高性能边缘云服务

AkrainoEdge Stack 是一个面向高性能边缘云服务的开源项目，并为边缘基础设施提供整体的解决方案。Akraino Edge Stack 致力于发展一套开源软件栈，用于优化边缘基础设施的网络构建和管理方式，以满足边缘计算云的要求

Open CORD 网络边缘基础设施

CORD 是为网络运营商推出的开源项目，旨在利用软件定义网络（SDN）、网络功能虚拟化（NFV）和云计算技术重构现有的网络边缘基础设施，并将其打造成可灵活地提供计算和网络服务的数据中心

Edgex foundry 面向物联网端的边缘计算开源平台

EdgeXFoundry 是一个面向工业物联网边缘计算开发的标准化互操作性框架，部署于路由器和交换机等边缘设备上，为各种传感器、设备或其他物联网器件提供即插即用功能并管理 它们，进而收集和分析它们的数据

可插拔的企业级 react 应用框架 umi

umi 的整个生命周期都是插件化的，甚至其内部实现就是由大量插件组成，比如 pwa、按需加载、一键切换 preact、一键兼容 ie9 等等，都是由插件实现

多维数组中加入变量名与坐标索引 xarray

xarray 是一个开源 Python 包，它可以使处理多维数组更加简单、高效并有趣。xarray 在原始类 NumPy 多维数组中引入了标签化的变量名称和坐标索引，实现了更直观、更简洁和更加不容易出错的能力

zTree v3插件

zTree 是一个依靠 jQuery 实现的多功能 “树插件”。优异的性能、灵活的配置、多种功能的组合是 zTree 最大优点 采用了 延迟加载 技术，上万节点轻松加载，即使在 IE6 下也能基本做到秒杀

bs2004/jquery.bsgrid

支持json、xml数据格式，皮肤丰富并且容易定制，支持表格编辑、本地数据、导出参数构建等实用便捷的功能，容易扩展，更拥有丰富的示例以及问题反馈的及时响应

Harris

JQuery TreeTable Plugin基于JQuery的表格树插件，按data-id和data-pid计算布局，在表格中展现树型结构。附加展开、关闭、查询、删除、注销、重置等公共函数

parky/domTemplate

一个基于jquery/zepto 非侵入式、不会破坏原来静态页面结构、可被浏览器正确显示的、格式良好的前端HTML模板引擎，动态模板文件不能被浏览器解释、必须要运行在服务器中才能显示出效果

笔心/jquery.datepicker

日期选择控件，依赖jquery，代码简单，小巧，只提供最基本的功能，可以自己进行二次开发，自定义样式，内置简单模板引擎，精简代码，去掉trigger配置，保持更新

robergroup/pdman

PDMan是一款开源免费的数据库模型建模工具，支持Windows,Mac,Linux等操作系统，是PowerDesigner之外，更好的免费的替代方案。他具有颜值高，使用简单的特点

Apache ShardingSphere

Apache ShardingSphere(Incubator) 是一套开源的分布式数据库中间件解决方案组成的生态圈，它由Sharding-JDBC、Sharding-Proxy和Sharding-Sidecar（规划中）这3款相互独立，却又能够混合部署

wenshao/druid
